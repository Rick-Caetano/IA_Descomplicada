• Criação e formatação de planilhas

    Exemplos: gerar uma planilha a partir de uma lista de itens, criar fórmulas, sumarizar vendas por mês.

    Prompt exemplo:

    “Crie uma planilha com colunas: Data, Produto, Quantidade, Preço; some as vendas por mês.”

  • Limpeza e organização de dados

    Exemplos: transformar nomes em Maiúsculas, ordenar alfabeticamente, remover duplicatas, padronizar endereços.

    Prompt exemplo:

    “Receba esta lista de nomes e retorne-a em ordem alfabética e com cada nome começando com letra maiúscula.”

  • Resumos e simplificação de documentos

    Exemplos: resumir relatórios longos, extrair pontos-chave de uma reunião.

    Prompt exemplo:

    “suma este relatório em 5 tópicos fáceis de entender.”

  • Escrita e resposta de e-mails

    Exemplos: escrever e-mails formais, sugerir respostas rápidas, adaptar o tom para diferentes públicos.

    Prompt exemplo:

    “Escreva um e-mail formal pedindo reembolso com tom educado e direto.”

  • Ajuda com finanças e investimentos

    Exemplos: organizar um orçamento mensal, explicar opções de investimento em linguagem simples, simular cenários (risco vs retorno).

    Prompt exemplo:

    “Explique em linguagem simples a diferença entre CDB, Tesouro Direto e ações.”

  • Criação e edição de imagens e layouts

    Exemplos: melhorar qualidade de imagem, remover texto de uma foto, gerar layout de post para redes sociais ou cartazes.

    Prompt exemplo:

    “Gerar um layout de cartaz A4 para evento comunitário com título 'evento beneficente' e cores 'azul e amarelo'.”

  • Geração de textos práticos

    Exemplos: contratos simples, cartas oficiais, recibos, roteiros curtos para vídeos; tudo adaptado ao seu tom e ao público.

    Prompt exemplo:

    “Gere um recibo simples para prestação de serviço com campos: nome, serviço, valor, data.”

  • Aprendizado e tutorial personalizada

    Exemplos: ensinar conceitos (história, matemática) com exemplos fáceis, criar exercícios para preparação em concurso público e corrigir respostas.

    Prompt exemplo:

    “Explique o básico de juros simples em 3 exemplos práticos.”

  • Transcrição e organização de áudio

    Exemplos: transformar gravações em texto, criar atas de reunião a partir da transcrição.

    Prompt exemplo:

    “Transcreva este áudio e gere um resumo com os 5 principais pontos.”

  • Automação de tarefas simples

    Exemplos: preparar textos recorrentes, organizar informações, criar um programa de computador para realizar determinada tarefa.

    Prompt exemplo:

    “Crie um programa de computador que automatize o envio de e-mails semanais.”

  • Assistência em saúde e bem-estar (informativa)

    Exemplos: explicar sintomas de forma simples, sugerir perguntas para levar ao médico.

    Prompt exemplo:

    “Explique de forma simples o que significa pressão arterial alta e quais perguntas devo fazer ao médico.”

    Importante: IA não substitui diagnóstico médico. Use como complemento e consulte profissionais.
